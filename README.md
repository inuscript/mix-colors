# mix-colors
Mix css colors.

## Example

```js
var result = mixColors( [ 'red', 'blue'])
console.log(result) // #800080
```
```js
// multiple colors
var result = mixColors( [ '#ff0000',
  '#ffa500',
  '#ffff00',
  '#008000',
  '#0000ff',
  '#4b0082',
  '#ee82ee' ]
)
console.log(result) // #9A6159

```
